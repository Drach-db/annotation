Ты профессиональный аннотатор POV-видео ручной работы. Проанализируй это видео и создай детальную временную аннотацию всех действий.

ВАЖНО: Сначала создай контролируемый словарь на основе того, что ты видишь в видео!

## СОЗДАЙ КОНТРОЛИРУЕМЫЙ СЛОВАРЬ

Внимательно изучи видео и определи:

### ГЛАГОЛЫ (что делается в видео):
Перечисли все уникальные действия, которые ты наблюдаешь. Например: взять, положить, нанести, удалить, повернуть, нажать, открыть, закрыть и т.д.

### ОБЪЕКТЫ (с чем взаимодействуют):
Перечисли все объекты и части, которые видны в видео. Например: руки, пальцы, инструменты, материалы, поверхности, детали и т.д.

### ИНСТРУМЕНТЫ (чем работают):
Перечисли все инструменты и приспособления из видео.

### МАТЕРИАЛЫ (что используют):
Перечисли все материалы, жидкости, покрытия и т.д.

### ЭТАПЫ ПРОЦЕССА:
Определи основные этапы работы в видео (например: подготовка, основная работа, завершение).

## ПОСЛЕ СОЗДАНИЯ СЛОВАРЯ - АННОТИРУЙ ВИДЕО

СТРУКТУРА ОПИСАНИЯ:
Используй шаблон: [ГЛАГОЛ] + [ОБЪЕКТ/ОБЛАСТЬ] + [ИНСТРУМЕНТ] + [ДЕТАЛИ]

Используй ТОЛЬКО те термины, которые ты определил в контролируемом словаре выше!

Уровни детализации:
- 80% аннотаций: ГЛАГОЛ объект с_инструментом
- 15% аннотаций: добавь детали в скобках
- 5% аннотаций: укажи руку когда критично

ПРАВИЛА ПРЕДЛОГОВ:
- с_ = инструмент (работать с_инструментом)
- на_ = цель/направление (нанести на_объект)
- из_ = источник (взять из_контейнера)
- для_ = цель/время (ждать для_результата)
- в_ = контейнер/состояние (поместить в_место)

ВРЕМЕННАЯ СЕГМЕНТАЦИЯ:
- Минимальная длительность действия: 2 секунды
- Максимальная длительность: 60 секунд
- Начало = момент движения руки к объекту (НЕ контакт)
- Конец = момент отпускания объекта ИЛИ начало движения к следующему

ИЕРАРХИЯ АННОТАЦИЙ:
Используй префиксы для разных уровней:
- ЭТАП: высокоуровневые фазы (3-5 минут)
- ДЕЙСТВИЕ: функциональные единицы (10-60 секунд)
- МИКРО: критически важные детали (2-10 секунд)

СПЕЦИАЛЬНЫЕ МАРКЕРЫ:
- [НЕАКТИВНО] = для разговоров/пауз
- ПЕРЕХОД = для смены этапов работы
- Указывай неудачные попытки: (неудачная_попытка), (исправление_ошибки)
- Связывай действия через ID когда нужно показать последовательность

ФОРМАТ ВЫВОДА:

=== КОНТРОЛИРУЕМЫЙ СЛОВАРЬ ===
ГЛАГОЛЫ: [список всех глаголов из видео]
ОБЪЕКТЫ: [список всех объектов]
ИНСТРУМЕНТЫ: [список инструментов]
МАТЕРИАЛЫ: [список материалов]
ЭТАПЫ: [список этапов]

=== ВРЕМЕННАЯ АННОТАЦИЯ ===
MM:SS-MM:SS ДЕЙСТВИЕ описание

Пример формата:
00:00-00:08 ВЗЯТЬ инструмент (начальная_подготовка)
00:08-00:22 РАБОТАТЬ с_объектом используя_инструмент
00:22-00:35 ПРОВЕРИТЬ результат_работы
00:35-03:45 ЭТАП: основная_фаза_работы
...

ВАЖНО:
1. Сначала ВСЕГДА создавай словарь на основе видео
2. Минимум 1-2 аннотации в минуту видео
3. Все термины соединяй подчёркиванием для машинной обработки
4. Следи за консистентностью терминологии из твоего словаря
5. Отмечай смены инструментов как ключевые точки
6. Включай неудачные попытки и исправления

Проанализируй видео, создай контролируемый словарь и затем создай полную временную аннотацию всех действий.
